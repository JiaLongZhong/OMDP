Browser Fingerprinting

How the Computer Identify the User:

  • • Firefox 72 is apparently browser fingerprint protection.
  • • A browser fingerprint is a fingerprint of your browser
  • • It's essentially a unique ID that is generated and it can be used to uniquely identify your browser’s life.
  • • The interactivity of the internet today is powered by JavaScript
  • • JavaScript can know the resolution of screen, and the number of pixels on the screen, and the orientation of your screen.
  • • When using a website, the web designer can put a hidden code that runs in the background that collects data about your device and preference.
  • • The website now knows what device you are using, what browser and operating system your device is using, the time the device is being used and the language your device is set to, plus others.
  • • It can grab information about your computer that you don't even know it's there.
  • • It can tell the amount of time you move or click with a mouse cursor.
  • • It can tell if you minimum or maximize your screen.
  • • The browser is drawing stuff in HTML canvas element that can be used to uniquely identify your computer
  • • A combination of bits is what makes a person unique.
  • • Google analytics is an example of a program that is used to track human interaction with the computer.
  • • Fingerprinting is a problem in terms of privacy
  • • Just with a few codes from the right people you can be able to identify billions of people by their browser fingerprint ID.
  • • Everyone out there is doing browser fingerprinting and practically.
  • • Which means that you can be using a certain browser even if you go into private browsing mode the number of bits that uniquely identify the unit doesn't even change and you can verify for yourself.
  • • Basically the only certain way to turn off browser fingerprinting is to physically turnoff JavaScript for the website.
