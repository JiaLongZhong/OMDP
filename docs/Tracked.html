How You are Tracked Online

Being Tracked:

  • 1. Every time we do anything online, there is always a serious risk.

  • 2. The risk being: user data are being collected online.

  • 3. Online advertising nowadays is unavoidable, and each time you click on an ad data will be collected by the government or companies.

  • 4. Apple, Google, Facebook are all well-known companies that track people's online data.

  • 5. Search engine like DuckDuckGo allow you to search the web without being tracked.

  • 6. The most common tools to collect information about us are known as cookies.

  • 7. Another tool that companies use is browser fingerprinting.

  • 8. Cookies – basically text files that hold information about the client computer. It helps the server identify the user. The server will send a set of cookies to the Browser and the browser stores the information in a local machine. Cookies are very useful; they allow the user's personal computer to remember the user username and password. It enhances web browsing experience, by showing comfortable websites that the user has used before and trusted. It also can be ignored, by sending an email of a similar item that was recently purchased. But be aware that data being stored in the cookies can be used against you.

  • 9. Sessions- is a temporary directory that holds more information than the cookie but is deleted when the user closes the browsers. Seasons are assigned to unique id that can be used to retrieve store data. When a session is created the cookie will store the unique session ID to the user computer. Basically, the session acts as a temporary cookie in the server side.

  • 10. Browser fingerprinting- JavaScript code to the browser, acts as a machine fingerprint that helps identify the user identity. It tracks all the applications and extensions in your device, your time zone, etc.

  • 11. VPN will help hide your device IP address, but it wouldn’t stop the data collection.
